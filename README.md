# PowerBIBasics

Interactive Power BI report built to analyze financial margins and profitability across different regions, products, and market segments.
**Data source:** Financial sample dataset
**Visual components:**
1. KPI Card:Tracking Total Profit.
2. Geospatial Analysis: Profit by Country (Map visual).
3. Trend Analysis: Profit by Date (Column chart).
4. Categorical Breakdown: Sales by Product and Segment (Clustered bar chart).
5. Interactive Filtering: Date slicer for dynamic timeline adjustments.
